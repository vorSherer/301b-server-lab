Node and npm  (I used node v10.19.0 and npm 6.13.4)
DEPENDENCIES
    "dotenv": "^8.2.0",
    "express": "^4.17.1"
    nodemon

#### `.env` requirements
* `PORT` - 3000
* `DATABASE_URL` - URL to the running mongo instance/db
* any environment variables like api keys as needed

#### Running the app
* `npm start`
